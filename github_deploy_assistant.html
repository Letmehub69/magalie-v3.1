Déploiement sur GitHub Pages

1

Crée le repository sur GitHub

Clique sur le bouton ci-dessous pour créer un nouveau repository.

Créer un nouveau repository

Paramètres recommandés:
• Owner: Letmehub69
• Repository name: fitmaster-pro
• Description: "FitMaster Pro - Application d'entraînement premium"
• Public
• NE PAS initialiser avec README, .gitignore, ou license

2

Télécharge les fichiers

Télécharge l'archive ZIP contenant l'application complète.

Télécharger fitmaster-pro.zip

Taille: ~60 KB • Contient toute l'application

3

Upload les fichiers sur GitHub

Une fois le repository créé, upload les fichiers:

  1. Extrais le fichier ZIP
  2. Va sur ton repository GitHub
  3. Clique sur "Add file" → "Upload files"
  4. Glisse tous les fichiers extraits
  5. Commit avec le message "Deploy FitMaster Pro"
4

Active GitHub Pages

Active GitHub Pages dans les settings du repository:

Activer GitHub Pages

Configuration:
• Source: Deploy from a branch
• Branch: main
• Folder: / (root)
• Clique sur Save

URL de ton application: https://letmehub69.github.io/fitmaster-pro

Déploiement via ligne de commande

Si tu préfères utiliser la ligne de commande, voici les commandes:

Copier
# Clone le repository (après l'avoir créé sur GitHub) git clone https://github.com/Letmehub69/fitmaster-pro.git cd fitmaster-pro
Copier
# Copie les fichiers de l'application cp /chemin/vers/workout_timer_final.html index.html cp /chemin/vers/README.md . cp /chemin/vers/netlify.toml . touch .nojekyll
Copier
# Commit et push git add . git commit -m "Deploy FitMaster Pro" git push origin main
