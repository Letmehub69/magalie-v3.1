# 🌙 Mission Sommeil

Guide complet pour aider les parents à guider leurs enfants (2-4 ans) vers l'autonomie du sommeil.

## Description

Cette page web présente une méthode complète et bienveillante pour transformer le coucher d'un enfant en un moment de calme et de sérénité. Elle est conçue pour être:

- **Responsive** : S'adapte parfaitement à tous les écrans (mobile, tablette, desktop)
- **Accessible** : Utilise des couleurs contrastées et une typographie lisible
- **Pédagogique** : Structure claire avec des étapes progressives
- **Apaisante** : Palette de couleurs douces (bleus, violets) pour une ambiance calme

## Fonctionnalités

- Design moderne avec animations subtiles
- Structure en 5 sections principales
- Tableaux et cartes interactives
- Palette de couleurs optimisée pour le thème du sommeil
- Compatible avec tous les navigateurs modernes

## Technologies utilisées

- HTML5
- CSS3 (Flexbox, Grid, Variables CSS)
- Font Awesome pour les icônes
- JavaScript léger pour les animations

## Comment utiliser

1. Ouvrez `mission-sommeil.html` dans votre navigateur
2. Suivez les 5 étapes présentées
3. Appliquez les conseils avec patience et constance

## Déploiement

Cette page peut être déployée sur:
- GitHub Pages
- Netlify
- Vercel
- Tout hébergeur statique

## Auteur

Guide créé avec soin pour aider les parents à retrouver des nuits paisibles.
