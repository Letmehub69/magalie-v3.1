📋 Comment utiliser l'application :

  • 1. Saisis ton exercice dans la colonne "Exercice"
  • 2. Indique le nombre de répétitions dans la colonne "Répétitions"
  • 3. Sélectionne le temps de repos souhaité (30s, 60s, 90s)
  • 4. Clique sur "Démarrer" pour lancer le compte à rebours
  • 5. Une notification sonore se déclenche à la fin du timer
  • 6. Passe à l'exercice suivant une fois le repos terminé
